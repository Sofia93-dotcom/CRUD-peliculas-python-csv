# CRUD de Películas en Python (CSV)

Proyecto en Python que implementa un sistema CRUD
(Crear, Leer, Buscar, Editar y Eliminar) de películas utilizando
archivos CSV como almacenamiento.

## Funcionalidades
- Agregar películas
- Mostrar películas
- Buscar por título (búsqueda parcial)
- Editar películas
- Eliminar películas
- Manejo de errores y validaciones

## Tecnologías
- Python
- csv
- Programación estructurada

## Ejecución
Ejecutar el archivo `peliculas_csv.py` desde la consola.
